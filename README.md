# UniGO
UniGO is a campus ride-booking app that helps students easily find or offer rides within the university. With UniGO, users can book or accept rides in just a few taps. It also supports cashless payments and a rewards system that can be redeemed for exciting perks. UniGO makes campus transportation convenient, safe, and rewarding.
